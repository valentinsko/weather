    Карта движения осадков — Яндекс Nowcast

    Прогноз радара от Яндекса. Если виджет не загрузился — откроется по ссылке.

    Источники: Open-Meteo (модели ECMWF, ICON-DWD, GFS, Météo-France, GEM, JMA), OpenWeatherMap, WeatherAPI, MET Norway. Карта осадков — yandex.ru/pogoda/maps/nowcast.

    Open source · MIT · github

    Настройки

    Источники без ключа

    Метеомодели через Open-Meteo. Включаются и работают сразу.

    Источники с ключом

    Ключи хранятся только в этом браузере (localStorage), на сервер не отправляются.

    Получить: openweathermap.org · 1M вызовов/мес бесплатно

    Получить: weatherapi.com · 1M вызовов/мес бесплатно

    Премиум-источники

    Эти сервисы требуют backend-прокси (CORS / JWT). См. README в репозитории — там описано, как добавить через Cloudflare Workers или Vercel Functions.

    • Google Weather API — Maps Platform, до 10k вызовов/мес бесплатно
    • Яндекс.Погода — нужен бизнес-аккаунт в Yandex Cloud
    • Apple WeatherKit — нужен Apple Developer ($99/год) и JWT-подписание
